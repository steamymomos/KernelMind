# KernelMind Midterm Submission-SoC 2026

This repository contains my midterm submission for the SoC-26 KernelMind project.

Completed work includes:
- Assignment 1: Process abstraction, POSIX API, fork/wait behavior, process states, context switching, and scheduling theory.
- Assignment 2: The Adrian Descent, a tabular Q-learning control problem with physics-based environment, stochastic wind, reward shaping, training loop, evaluation, plots, metrics, and report.
- Assignment 3: Hybrid Meta-Scheduler, an RL-based CPU scheduling meta-agent that dynamically selects between FCFS, SJF, and Round Robin policies and compares performance against static baselines.

The repository includes notebooks, Python scripts, trained Q-tables, reports, plots, and evaluation metrics for reproducibility.
